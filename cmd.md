# Подсказки по командной строке

 Команда смены директории
 ```sh
cd c:/folder_name 
 ```

 Команда отображения текущей директории. Mac0s, Linux
 ```sh
 pwd       
 ```

Листинг тущей директории
Windows:
```sh
dir
```
Linux,MaxOs:
```sh
ls
```

Удаление файла в Windows:
```sh
del <file name>
```
в Linux, MacOs:
```sh
rm <filename>
```
